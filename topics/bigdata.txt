MapReduce — это фреймворк для вычисления некоторых наборов распределенных задач с использованием большого количества компьютеров (называемых «нодами»), образующих кластер.
Hadoop - від Apache
	основні компоненти:
		- Hadoop Distributed File System (HDFS) - розподілена файлова система, яка дозволяє зберігати дані величезного об"єму
		- Hadoop YARN - фреймворк для управління ресурсами кластера і менеджмента задач (в тому числі MapReduce)
		- Hadoop common
	Крім того є багато інших компонентів:
		- Hive інструмент для перетворення sql запитів в MapReduce задачі
		- Pig - мова програмування для аналізу даних на високому рівні
		- Cassandra – високопродуктивна розподілена key-value база даних
		- ZooKeeper - сервіс для розподілного зберігання конфігурацій і синхронізації змін конфігурацій
		- Mahout - machine learning
