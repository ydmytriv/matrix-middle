Git, CodePlex, Google Code, HG, SourceForge, SVN, TFS
Github
	+ найбільше юзерів
	+ найршоиреніший
	+ пошук коду в публічних репо
	+ Bug tracking
	- щоб користуватись всіма можливостями, потрібен платний акаунт
	- Обмеження в розмірах (репозиторій до 1 ГБ, файл до 100 Мб)

Gitlab
	+ безкоштовниі приватні репо
	+ opensource
	+ Bug tracking mechanism
	+ Можна редагувати файли прямо в веб-інтерфейсі
	+ інтеграція з LDAP
	- повільний інтерфейс
	- Часті технічні проблеми з репо

BitBucket
	+ безкоштовний для невеликих команд (1-5 чол)
	+ Query management system
	+ автентифікація через github і соц. мережі
	+ інтеграція з джирою
	+ імпорт існуючих репо
	- немає пошуку 
	- не такий стабільний як GitHub
